# Alpinesight

- This project uses the vercel template called: `https://vercel.com/templates/ai/ai-sdk-python-streaming` as the starting point, which comes with a simple LLM chat application.

# Local Development

- You need to have a `alpinesight-app/.env.local` (see `.env.local.template` for a sample of it)

- cd alpinesight-app && pnpm dev
